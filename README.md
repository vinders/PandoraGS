PandoraGS is a GPU driver plugin for PlayStation emulators (PSEmu compatible).

    Author       : Romain Vinders
    Build        : Visual Studio 2017+ / gcc 7.1+
    Requirements : OpenGL 4.1+ compatible drivers
                   256MB+ VRAM
